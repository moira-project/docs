### 개요

**사적** 모임 일정 관리 서비스

### 용어 정리

- 사용자 (user)
- 멤버 (member) : 모임의 구성원
- 모임장 (leader) : 모임의 리더 . 모임을 생성한 사람.
- 메인 캘린더(main calendar) : 메인 페이지에 배치되어 있음. 속해 있는 그룹들에서 생성된 이벤트들을 한 번에 볼 수 있음.(이벤트 모아보기)
- 모임(group)
- 모임 캘린더(group calendar) : 모임 별 캘린더
- 이벤트(event) : 모임 안에서 생성됨
- 이벤트 할일 목록(event todo) : 이벤트를 위하여 사전에 필요한 할 일 목록
  - 할일 toss (todo toss) : 할당된 이벤트 할일 목록을 다른 멤버에게 toss (알림 필요)
- 이벤트 설문 폼(event form) : 이벤트를 위해 사전에 정해야 할 요소들에 대한 멤버들의 의견을 취합하기 위한 수단.
- 이벤트 알림: 이벤트에 대한 알림
  - 이벤트 참여자 추가 알림
  - 이벤트 참여자 삭제 알림
  - 이벤트 todo toss 알림
  - 이벤트 삭제 알림
  - 이벤트 추가 알림
  - 이벤트 내용 변경 알림

### 기능

1. 사용자 CRUD
2. 전체 event 모아 보기 (메인 캘린더)
3. 모임 CRUD
4. 모임 별 캘린더(오늘의 todo(해당 이벤트와 매칭 되어 있음.), 이벤트 일자 표시)
5. 그룹 이벤트 CRUD , 멤버 간 공유(멤버 모두에게 수정 권한 부여)
   1. 이벤트 알림 기능
      - 이벤트 내용 변경 알림
      - 이벤트 추가 알림
      - 이벤트 삭제 알림
      - 이벤트 todo toss 알림
      - 이벤트 참여자 삭제 알림
      - 이벤트 참여자 추가 알림
   2. 그룹 이벤트 todo 관리 기능(그 일을 언제 할 지 선택할 수 있고, 해당 이벤트와 연결)
   3. 그룹 이벤트 todo 역할 배정.
   4. **그룹 이벤트 todo 다른 멤버에게 toss**
   5. 그룹 이벤트 참여자 관리
   6. 그룹 이벤트를 위한 설문 폼
6. 사용자 생일 캘린더 상에 자동 표시(메인 캘린더, 그룹 캘린더 모두)
